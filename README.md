# RaxKernel — Ядро операционной системы RaxOS

![License](https://img.shields.io/badge/license-GPL--3.0-blue.svg)
![Status](https://img.shields.io/badge/status-in--development-orange)

**RaxKernel** — это сердце независимой операционной системы **RaxOS**, которая создается с нуля силами сообщества энтузиастов при активной поддержке искусственного интеллекта. 

Проект RaxOS позиционирует себя как свободная ОС, ориентированная на мнение пользователей, конфиденциальность и прозрачность разработки.

---

## 📌 О проекте

* **Независимость:** Собственная база кода, разрабатываемая с акцентом на технологический суверенитет.
* **ИИ-Ассистирование:** Уникальный подход к проектированию и оптимизации архитектуры ядра с привлечением современных нейросетевых моделей.
* **Открытость:** Частично открытый исходный код (компоненты ядра полностью открыты) и управление, основанное на голосовании сообщества.

---

## 💻 Технологический стек
* **Язык разработки:** C / C++
* **Архитектура:** x86_64
* **Сборка:** Make
